Using cvedata#

Install#

Development#

git clone https://github.com/clearbluejar/cvedata
pip install .

Production#

pip install https://github.com/clearbluejar/cvedata/releases/download/v0.2.2/cvedata-0.2.2-py3-none-any.whl

Generate Release Data#

python -m cvedata.update

Download Release Data#

Instead of generating data, you can download the latest release data generated for the particular version.

python -m cvedata.download

This will unpack the data to your local cvedata installation in the data folder.
